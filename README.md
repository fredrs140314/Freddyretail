# Brand Domain Analyzer V2

## Features

- Compare multiple domains
- Analyze brand mentions
- Improved sitemap detection
- Traffic comparison graph
- Streamlit dashboard

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Structure

- app.py → frontend/UI
- backend.py → crawling and analysis
- requirements.txt → dependencies
- README.md → documentation

Domains:
- ica.se
- coop.se
- willys.se
