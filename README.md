# Brand Product Page Analyzer - Single File Version

This version avoids backend import errors by putting the full app in `app.py`.

## Files

```txt
app.py
requirements.txt
README.md
.gitignore
```

## Install

```bash
pip install -r requirements.txt
```

## Run locally

```bash
streamlit run app.py
```

## Streamlit Cloud

Upload only these files to your GitHub repo:

- app.py
- requirements.txt
- README.md
- .gitignore

Delete old files like:

- backend.py
- old app.py versions
- old folders

Set Streamlit main file path to:

```txt
app.py
```
